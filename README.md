# AtividadeEmbaca06_02
 Repositorio para alocação da atividade do Embarcatech 06/02/2025
O código fará o LED piscar com um efeito de fade in e fade out.
Se quiser um brilho fixo, o usuário pode chamar set_brightness(valor) com um número entre 0 (apagado) e 255 (máximo).
Se quiser mudar o GPIO, basta alterar #define PWM_GPIO 15 para outro pino suportado por PWM.
